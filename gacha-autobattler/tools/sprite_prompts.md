# AI Sprite Generation Prompts

All prompts are designed for video generation (5-12 seconds recommended).
- **Resolution**: 1:1 square aspect ratio, 512x512 or 1024x1024
- **Camera**: Fixed, no movement
- **Background**: Solid color (for easy removal) or transparent if supported
- **Style**: Fantasy game character, stylized/semi-realistic

## File Naming Convention

When saving videos, use this naming format:
- `{unit_id}_{animation}.mp4`

Examples:
- `fire_imp_idle.mp4`
- `fire_imp_attack.mp4`
- `fire_imp_hurt.mp4`
- `water_sprite_idle.mp4`
- etc.

Place all videos in: `gacha-autobattler/tools/`

## Batch Conversion

Once all videos are generated, run the sprite sheet maker for each:
```bash
python sprite_sheet_maker.py fire_imp_idle.mp4 --frames 8 --output ../assets/units/ai_sprites/fire_imp/idle.png
python sprite_sheet_maker.py fire_imp_attack.mp4 --frames 6 --output ../assets/units/ai_sprites/fire_imp/attack.png
python sprite_sheet_maker.py fire_imp_hurt.mp4 --frames 4 --output ../assets/units/ai_sprites/fire_imp/hurt.png
```

Or create all at once with a batch script (will be provided after videos are generated).
